Emergency Ambulance Traffic Coordination System

An intelligent system that reduces ambulance response time by making traffic infrastructure emergency-aware. It predicts route conflicts, avoids ambulance deadlocks, activates green corridors at intersections, and performs priority-based optimal rerouting using real road geometry.

▶ How to Run

Open index.html in a browser

Select start & destination for Ambulance A and B

Set priorities (1 = critical)

Click Run Simulation

✨ Features

Real road routing (OSRM)

Priority-based traffic control

Green corridor simulation

Deadlock avoidance between ambulances

AI-assisted optimal rerouting (cost-based)

Explainable decision logic
